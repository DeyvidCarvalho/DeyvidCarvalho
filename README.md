💼 Projeto em destaque
Flask Inventory Dashboard  
Sistema web para inventário de TI com foco em produtividade operacional.

Funcionalidades principais:

- Login e sessão
- Aprovação de usuários por administrador
- Dashboard por filtros de RAM, CPU e Windows
- Listagem completa de computadores
- Tela de detalhe com histórico de mudanças
- Busca por usuário, computador, serial e IP
- Relatórios técnicos com exportação CSV
- Banco fictício para demonstração
- Melhorias de usabilidade (atalhos, sugestões inteligentes de login, persistência local)
